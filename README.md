# Windows-Server-MCSA-Lab
Hands-on Windows Server lab with Active Directory, DNS, DHCP, GPO, and PowerShell automation.

## Overview
This project demonstrates hands-on implementation of a Windows Server
enterprise environment.

## Key Components
- Active Directory (Primary & Secondary DCs)
- DNS (Forwarders, Conditional Forwarders, Stub Zones)
- DHCP with scopes, exclusions and failover
- Group Policy hardening
- File Server and NTFS permissions
- PowerShell automation
